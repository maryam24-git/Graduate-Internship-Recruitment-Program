# Graduate & Internship Recruitment Program – MS Project 2016

## Project Overview

This repository contains the complete project management schedule for a **Graduate and Internship Recruitment Program** developed using **Microsoft Project 2016**.

The project demonstrates the planning, recruitment, selection, onboarding and orientation processes required to successfully hire and integrate graduates and interns into an organization.

## Objectives

* Attract qualified graduates and interns.
* Streamline the recruitment process.
* Improve candidate experience.
* Reduce hiring time.
* Ensure effective onboarding and training.

## Project Management Tool

* Microsoft Project 2016

## Project Phases

### Phase 1: Program Planning & Design

* Define Program Objectives and Scope
* Identify Departments and Roles
* Define Competencies and Role Profiles
* Budget and Headcount Approval
* Internship Structure Design

### Phase 2: Employer Branding & Marketing

* Develop Recruitment Branding
* Create Job Descriptions
* University Partnership Outreach
* Job Portal and Social Media Posting

### Phase 3: Application & Screening

* Application Collection
* CV Screening and Shortlisting
* Online Assessment Tests
* Candidate Shortlist Finalization

### Phase 4: Interviews & Selection

* Interview Scheduling
* HR Interviews
* Technical Interviews
* Candidate Evaluation
* Offer Approval

### Phase 5: Offer & Onboarding Preparation

* Offer Letter Distribution
* Candidate Follow-up
* Documentation Collection
* IT and Workstation Preparation
* Onboarding Planning

### Phase 6: Onboarding & Orientation

* Company Orientation
* Department Induction
* Mentor Assignment
* Policy and Compliance Training
* Role-Specific Training
* Goal Setting
